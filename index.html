Primeira Aula de HTML - Tags de Texto

Olá, pessoal! Nesta aula, vamos aprender sobre as tags de texto mais comuns e como elas são usadas para estruturar o conteúdo em uma página web.

Tags de Títulos

As tags de título vão de <h1> (o mais importante) a <h6> (o menos importante). Elas definem a hierarquia do conteúdo.

Tags de Formatação e Ênfase

Essas tags servem para dar importância, ênfase ou outro tipo de formatação ao texto.

<b> e <strong>: Texto em negrito e texto importante.
<i> e <em>: Texto em itálico e texto com ênfase.
<u>: Texto sublinhado.
<small>: Texto menor<0/small>.
<mark>: Texto marcado.
<del>: Texto deletado ou excluído.
<ins>: Texto inserido.
<sup> e <sub>: Texto com sobrescrito e com subscrito (bom para fórmulas: H2O).

Texto Pré-formatado (<pre>)

O <pre> preserva todos os espaços e quebras de linha que você digitar, sendo ótimo para exibir código ou texto formatado:

function mostrar() {
    // Note que este código
    //               mantém o espaçamento
    //      exatamente como digitado.
    console.log("Olá!");
}

Tags de Listas

Lista Não Ordenada (<ul> - Unordered List)

Usada para itens sem uma ordem específica (geralmente com marcadores).

  • Item um (usando a tag <li>)
  • Item dois
  • Item três

Lista Ordenada (<ol> - Ordered List)

Usada para itens em uma sequência específica (numerada ou com letras).

  1. Primeiro passo
  2. Segundo passo
  3. Terceiro passo
